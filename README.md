# HX Bucket for Scoop

[![Tests](https://github.com/bgzo/hx-scoop/actions/workflows/ci.yml/badge.svg)](https://github.com/bgzo/hx-scoop/actions/workflows/ci.yml) [![Excavator](https://github.com/bgzo/hx-scoop/actions/workflows/excavator.yml/badge.svg)](https://github.com/bgzo/hx-scoop/actions/workflows/excavator.yml)

Self use bucket for myself.

## Get Started

```powershell
scoop bucket add hx https://github.com/bgzo/hx-scoop
scoop install hx-scoop/<manifestname>
```
